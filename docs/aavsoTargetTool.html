AAVSO Target Tool

AAVSO provides a list of objects of interest via a system called "Target tool".

This list shows observations that professional astronomers would like assistance with in the form of additional observations, many of which can be accomplished by amateur astronomers.

In order to use the target tool, you must create an account.

The yellow text objects are the alerts from the target tool

When zoomed in, the alert number appears as well.

Time navigation

SkyTravel allows you to change the time of the current view.

Key Action
( / ) -/+ year
{ / } -/+ month
[ / ] -/+ day
< / > -/+ hour
0 / 9 -/+ minute
@ Reset to current time
